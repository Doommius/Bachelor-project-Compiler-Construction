## How to run

### Compile entire project
To make the entire project:

```
make
```

### Compile specific parts
To compile main program:

```
make compiler
```

To compile tests
```
make tests
```

### Run the program(s)

For compiler:
```
./compiler
```

For tests:
```
./tests
```

## How to clean-up project
Clean only build files
```
make clean
```

Clean build files and compiled programs
```
make clean-all
```
